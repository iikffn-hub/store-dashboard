# لوحة إدارة المتجر - نسخة ويب

نسخة ويب أولية مبنية لتكون قاعدة لنظام يعمل من أي جهاز **بعد رفعه على استضافة أو سيرفر**.

## المميزات
- تسجيل دخول
- لوحة تحكم رئيسية
- إدارة الطلبات
- إدارة العملاء
- إدارة بنك الإيميلات
- إدارة الاشتراكات
- قاعدة بيانات SQLite محلية
- استيراد مبدئي للبيانات من ملفات Excel التي رفعتها

## بيانات الدخول الافتراضية
- اسم المستخدم: `admin`
- كلمة المرور: `Store@12345`

## التشغيل المحلي
```bash
pip install -r requirements.txt
python app.py
```
ثم افتح:
```bash
http://127.0.0.1:5000
```

## التشغيل من أي جهاز
لازم ترفع المشروع على:
- Render
- Railway
- VPS
- أي استضافة تدعم Python/Flask

بعد الرفع سيصير عندك رابط عام وتقدر تدخل من الجوال أو الكمبيوتر.

## ملاحظات مهمة
- القاعدة الحالية SQLite، مناسبة كبداية ومشروع صغير.
- لو تبي استخدام أقوى ومتعدد المستخدمين، يفضّل لاحقاً تحويلها إلى PostgreSQL أو MySQL.
- كلمة المرور حالياً محفوظة بشكل بسيط لتسريع النسخة الأولية. في النسخة الاحترافية الأفضل استخدام تشفير كلمات المرور وصلاحيات مستخدمين.
